# 🚀 Antigravity Patcher

Патч для Google Antigravity IDE, снимающий все ограничения и работающий с **любым** Google‑аккаунтом.  
Больше никаких проверок email — полный доступ к функциям IDE без VPN и дополнительной верификации.

## ⚙️ Установка Antigravity

1. Скачайте официальную версию Antigravity с сайта:  
   👉 [https://antigravity.google](https://antigravity.google)
2. Установите приложение со стандартными настройками (путь установки менять **не нужно**).
3. **Не запускайте IDE после установки!**  
   Сначала примените патч.

> ⚠️ Если вы уже запускали Antigravity — закройте его полностью перед патчем.

## 🔧 Применение патча

### Windows

Запустите **PowerShell от имени администратора** и выполните:

```powershell
cd $env:USERPROFILE\Downloads
powershell -ExecutionPolicy Bypass -File .\windowsag.ps1
```

### macOS

Откройте **Терминал** и выполните:

```bash
cd ~/Downloads
chmod +x macosag.sh
xattr -cr macosag.sh
./macosag.sh
```

### Linux

Откройте **Терминал** и выполните:

```bash
cd ~/Downloads
chmod +x linuxag.sh
sudo ./linuxag.sh
```

> Если файл скачан в другую папку, укажите правильный путь в команде `cd`.

## 🛠️ Решение возможных ошибок

- **Устанавливайте Antigravity по стандартным путям** — скрипт ожидает найти файлы в директории по умолчанию.
- **После автоматического или ручного обновления IDE патч слетит.**  
  Просто повторите процедуру для вашей ОС — скрипт снова применит все необходимые изменения.

## 🔍 Как это работает

Скрипт находит основной файл `main.js` в установленной версии Antigravity и внедряет в него модифицированную функцию аутентификации.

## ❓ Часто задаваемые вопросы

### Нужен ли VPN?
**Нет.** После применения патча Antigravity работает напрямую, без VPN.

### Можно ли использовать любой Google‑аккаунт?
**Да,** ограничений по email нет.

### Безопасно ли это?
Патч модифицирует только локальные файлы приложения и **не отправляет** никаких данных вовне. Рекомендуется использовать только официальные сборки Antigravity.
